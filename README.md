KeithleyControl Support Keithley Powersupply control by LAN Interface.
基于WPF框架，MVVM模型开发，Keithley 电源控制，支持LAN口远程控制。

## 功能列表

- [x] 电源控制基本功能（电压、电流、实时电流曲线）
- [x] 支持LAN口远程控制
- [x] 支持设置电压，电流
- [x] 支持实时获取电压电流值
- [x] 采用oxyplot实时绘制电流曲线
- [ ] GPIB控制（待开发）


## 构建

- [x] VisualStudio 2022（基于 .NET WPF框架验证）
